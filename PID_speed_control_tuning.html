Internal model control simulations for autonomous cars and rovers¶

V. Venkatesh¶

In this work, we design a "PID" controller based on internal model control (IMC) tuning rules. Numerical solutions to the equation of motion are employed to compute proportional and integral gains. The differential equation solver in Scipy forms the crux of the methodology herein. The approach involves solving for acceleration balance according to
$\Sigma F = m\frac{dv}{dt}$

The forces considered herein are drag and car/rover momentum as below
$\frac{dv}{dt} = \frac{1}{m} \Big(F_p u - \frac{1}{2}C_d \rho A v^2\Big)$

where

Variable Meaning
m Mass of car/rover (kg)
v Velocity (m/s)
t Time (s)
$C_d$ Drag coefficient
$\rho$ Density of air (km/m^3)
A Cross-sectional area
$F_p$ Thrust coefficient in Newton/input

# Define a function that computes acceleration for a car
def car(v, t, u, load):
    # Inputs
    # v - Speed of car in m/s
    # t - Time in seconds
    # u - Drive control signal fed to electric motor
    # load - Weight of passengers in the car
    Cd  = 0.24     # Drag coefficient
    rho = 1.225    # Density of air in kg/m^3
    A   = 5.0      # Cross-sectional area in m^2 that cuts through wind during motion
    Fp  = 30.0     # Thrust parameter in N/V
    m   = 500.0    # Mass of car in kg

    # Calculate acceleration
    dv_by_dt = (1.0/(m+load)) * ((Fp * u) - (0.5*rho*Cd*A*(v**2)))
    return dv_by_dt

# Initialization
tf = 300
nsteps = 301
delta_t = tf / (nsteps-1)
ts = np.linspace(0, tf, nsteps)

        u = -50.0
    v = odeint(car, v0, [0, delta_t], args=(u, load))

Proportional gain
$K_p = \frac{\Delta{y}}{\Delta{u}}$ $=\frac{100}{25}$=4

Response delay at start
Delay between start of output $v(t)$ and input $u(t)$
$\theta_p$ = 11

Response time constant
Time taken to reach $(1 - \frac{1}{e})*v_{max}(t)$
$\tau_p = 11.5$

We use the above coefficients to design an autonomous control system defined by
$e(t) = Set_{point} - y(t)$

The input signal $u(t)$ is synthesized using $e(t)$ using the "PID" relationship below
$u(t) = u_{bias} + K_c e(t) + \frac{K_c}{\tau_i}\Sigma_{i=1}^{N}e_i(t) \Delta{t} + K_c \tau_D \frac{e_i(t) - e_{i-1}(t)}{\Delta{t}}$

where $K_c = \frac{1}{K_p} \Big(\frac{\tau_p}{\theta_p + \tau_c}\Big)$ and $\tau_i = \tau_p$

Internal Model Control (IMC) tuning correlation rules determine the overarching range of $\tau_c$

Philosophy $\tau_c$ rule
Aggressive max($0.1\tau_p$,$0.8\theta_p$)
Moderate max($\tau_p$,$8\theta_p$)
Conservative max($10\tau_p$,$10\theta_p$)

Equation of motion simulations for objects with large mass show dead-time to be neglegibile. Further, since $\tau_p \approx \theta_p$, the PID coefficients reduce to
$K_c = \frac{1}{\tau_p}$
$\tau_i = \tau_p$

For autonomous cars, we use the results shown above to obtain
$\tau_i = 11.5$
$K_c = \frac{11.5}{\lambda \tau_c}$

We run further simulations to investigate performance corresponding to $K_c = 0.833$ for autonomous cars.

For small rovers, simulations show that dead-time is significant. The internal model control choice is made according to
$K_c = \frac{\tau_p + 0.5 \theta_p}{\lambda K_p}$
$\tau_i = \tau_p + 0.5 \theta_p$

$K_c = \frac{4.4375}{\lambda}$
$\tau_i = 17.75$

Differential gains are omitted altogether in this work because of its increased sensitivity to noise.
